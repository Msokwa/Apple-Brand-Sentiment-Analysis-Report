# Apple-Brand-Sentiment-Analysis-Report
Overview 

This project performs a sentiment analysis on public opinion toward Apple Inc. using social media data from Twitter. The objective is to evaluate brand perception, identify customer pain points, and discover opportunities to improve brand engagement and marketing strategies.
Project Details
Data Source: Twitter
Time Frame: December 1–10, 2014 
Dataset Size: 3,886 tweets
Methodology: The analysis was conducted using Python, leveraging the pandas library for data handling, and VADER and TextBlob for sentiment evaluation.

Key Findings
Sentiment Distribution
The overall sentiment regarding Apple during this period leaned neutral.

Neutral: 57.7% (2,244 tweets) 
Negative: 31.4% (1,219 tweets) 
Positive: 10.9% (423 tweets) 

Insights
Positive Drivers: Sentiment was bolstered by keywords such as "iPhone," "MacBook," "smooth," and "performance".

Pain Points: Negative sentiment was primarily driven by product-related issues, specifically mentions of "battery," "slow" performance, "crash" events, and pricing concerns.

Recommendations
Based on the analysis, the report suggests the following actions to enhance the brand experience:
Event Monitoring: Actively track sentiment during major company events.
Address Product Issues: Prioritize addressing concerns regarding battery life and pricing.
Customer Support: Improve support services during weekends.
Targeted Promotion: Align marketing campaigns with periods of positive sentiment.
Expand Scope: Extend monitoring efforts to cover platforms beyond Twitter.
